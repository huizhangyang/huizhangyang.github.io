# 项目状态

- 项目：`huizhangyang.github.io` 静态 GitHub Pages 页面。
- 当前主页：`index.html` 为中文母版，`index_en.html` 为英文版。
- 资源目录：`SAR技术成果展示版_assets`，该目录图片由用户手工修订，后续不得重新生成或覆盖。
- 页面能力：保留主题切换、语言切换和基于 IP 的中英文自动切换；中国、新加坡默认中文，其它地区默认英文，手动选择写入 `localStorage.sar-language`。
- 英文页要求：必须以当前 `index.html` 新版文本为依据翻译，不使用旧版“客户交流”“Electronic Countermeasures”“Achievements”等口径；总标题使用 `Radar Imaging / Sensing / Countermeasures`。
