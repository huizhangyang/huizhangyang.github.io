# 英文页响应式标题与窗口自适应修正

## 背景

用户指出英文版页面在窗口变化时有固定尺寸感，右侧顶部大标题需要在一定范围内自适应缩小并保持单行。

## 修改内容

- 仅修改 `index_en.html`，不改中文页和图片资源。
- 将两栏布局中的侧栏宽度从固定 `304px` 改为 `clamp(248px, 22vw, 304px)`。
- 将英文 hero 标题改为基于容器查询单位 `cqw` 的自适应字号：`clamp(28px, 4cqw, 54px)`。
- 去掉标题固定 `max-width: 960px`，改为跟随右侧容器宽度。
- 在 1180px 以下切换为单栏布局，在 960px 以下允许标题自然换行，避免小窗口横向溢出。

## 验证

- 使用本机 Chrome headless 对 `index_en.html` 在 1440、1366、1280、1200、1180、1100、1024、960、768、560、390 宽度下测量。
- 1440 至 1024 宽度下英文标题保持单行且不溢出容器。
- 所有测试宽度均无页面横向溢出。
- 中英文页均保持 16 项成果、31 张图片，图片引用无缺失；主题切换、语言切换和 IP 自动切换仍存在。
